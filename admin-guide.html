First-Time Configuration

1. Advisor Settings Configuration

Navigate to Advisor Settings and configure:

  • Coach Email: Where student messages are sent (e.g., coach@yourdomain.com)
  • Meeting Link: Your Calendly or scheduling URL
  • Section Title: "College and Career Coach"
  • Statistics: Update student numbers and success rates
  • Recommendation Settings: Form links and email addresses

2. Test System Functionality

  • Register a test student
  • Upload a test document
  • Send a test message
  • Verify email delivery

Attendance Tracking

Recording Daily Attendance

Step 1: Navigate to Attendance
Step 2: Select date using date picker
Step 3: Mark each student as Present, Absent, Tardy, or Excused
Step 4: Click Save Attendance

Messaging Center

Two Communication Systems

1. Portal Messages (Internal Chat)

  • Direct communication between admin and students
  • Select student from list to view conversation
  • Send replies and follow-ups
  • Real-time message status tracking

2. Coach Messages (Email System)

  • Students send messages to career coach via email
  • Messages logged in admin portal for tracking
  • Coach email configured in Advisor Settings
  • View all coach communications in one place
Coach Email Setup: The coach email address is displayed at the top of the Coach Messages tab. Students can see exactly where their messages are being sent.

Document Management

Document Approval Workflow

Step 1: Students upload documents through their portal
Step 2: Documents appear in your Documents section
Step 3: Review document content
Step 4: Choose action: Approve, Send Back, or Delete

Community Service Letters (Special Process)

  1. Admin uploads 50-hour completion letter (status: Pending)
  2. Letter is invisible to student until approved
  3. Ms. Parkman reviews and approves letter
  4. Student sees download banner in their portal
  5. Provides complete auditable approval trail

Challenge System

Creating Challenges

Step 1: Navigate to Challenges
Step 2: Click Add New Challenge
Step 3: Fill in challenge details
Step 4: Link to roadmap step (optional)
Step 5: Set difficulty level and target percentage

Difficulty Levels

  • Grade-level agnostic: Suitable for all students
  • Freshman Pace: 9th grade appropriate
  • Sophomore Pace: 10th grade appropriate
  • Junior Pace: 11th grade appropriate
  • Senior Pace: 12th grade appropriate

Troubleshooting

Common Issues & Solutions

Database Table Missing Error:
Contact your web developer to resolve database table issues.
Cannot Access Admin Portal:
1. Check URL: https://yourdomain.com/portal-admin/
2. Verify login access with web developer
3. Clear browser cache
4. Contact technical support
File Upload Problems:
1. Contact web developer to check file permissions
2. Verify file size limits
3. Try different file formats
Coach Messages Not Working:
1. Configure coach email in Advisor Settings
2. Contact web developer to check email configuration
3. Test with different email provider
